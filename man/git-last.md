git-last(1) -- Display last x number of commits
================================

## SYNOPSIS

`git-last` &lt;NUMBER&gt;

## DESCRIPTION

Displays last `X` number of commits.
If no number is provided, return the last commit only.

## OPTIONS

&lt;NUMBER&gt;

## EXAMPLES

    $ git last 4
